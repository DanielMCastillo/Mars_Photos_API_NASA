Project

NASA Mars Rover API

This project consumes NASA's public API to retrieve real photos taken by Mars rovers. Users can select a specific date to view images captured on that day, along with details about the rover and the camera used.

Technologies Used

Built using Django for the backend and Bootstrap for a responsive design. It also uses the Fresco library to display an interactive gallery of images. The project incorporates templates, forms, and clean view and route management.

Purpose

The main goal of this project is to practice integrating third-party REST APIs, improve handling of external data, and deliver an engaging user experience focused on space exploration.
